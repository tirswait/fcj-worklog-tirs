Summary Report: “AI Agents, Prompt Engineering, and AIoT Projects on AWS”

Date & Time14/03/2026

Event Objectives

  • Understand how AI Agents extend traditional LLM capabilities
  • Learn prompt engineering techniques to improve output quality and efficiency
  • Explore real-world AIoT architectures using AWS services
  • Gain practical ideas for building AI-powered cloud applications

Speakers

  • Banh Cam Vinh – Topic: Building AI Agents with Strands
  • Nguyen Tuan Thinh – DevOps Engineer, First Cloud AI Journey
    Topic: Automated Prompt Engineering
  • Aiden Dinh – Operation Engineer, Katalon
    Topic: AIoT Projects

Key Highlights

AI Agents as the Next Step of AI Systems

  • Traditional LLMs → Single-step responses
  • AI Agents → Multi-step reasoning, tool usage, and decision-making
  • Better suited for real-world workflows requiring automation

Building AI Agents with Strands

  • Core elements: system prompts, tool integration, knowledge base
  • Workflow: input → reasoning → tool execution → response
  • Simplifies development compared to manual orchestration

Prompt Engineering and Optimization

  • Important components: role, task, context, constraints, output format
  • Best practices:
    • Write clear and structured prompts
    • Avoid ambiguity
    • Separate logical sections
  • Advanced techniques: Chain-of-Thought, RAG, role prompting
  • Better prompts → improved accuracy and lower cost

AIoT Architecture in Practice

  • Use case: Locker Management System
  • Hardware: Raspberry Pi, Arduino, RFID, sensors, camera
  • AWS services:
    • IoT Core → device communication
    • Lambda → processing logic
    • DynamoDB → storage
    • Rekognition → identity recognition
  • Goal: automate operations and improve efficiency

Key Takeaways

Shift Toward Agent-based AI

  • AI is moving from prompt-based interaction to agent-driven systems
  • Agents enable automation, reasoning, and integration

Prompt Design Matters

  • Prompt engineering directly affects output quality and cost
  • Well-designed prompts improve consistency and reliability

AWS Enables AIoT Systems

  • AWS provides a full ecosystem for integrating AI with IoT
  • Supports scalable and production-ready solutions

Applying to Work

Applying AI Agents

  • Use agents for automation workflows such as log analysis and report generation
  • Build multi-step AI features in system design

Improving AI Output

  • Apply structured prompting to improve response quality
  • Reduce ambiguity and optimize token usage

System Architecture Design

  • Apply AIoT architecture concepts to future projects
  • Integrate multiple AWS services into a unified system

Event Experience

Attending this event provided both conceptual understanding and practical insights into modern AI systems.

Learning New Concepts

  • Gained a clearer understanding of AI Agents and their capabilities
  • Recognized the importance of prompt engineering in real applications

Real-world System Perspective

  • AIoT project demonstrated how hardware and cloud services work together
  • Provided a clear view of building end-to-end systems

Practical Tools and Techniques

  • Explored Strands for building AI Agents
  • Learned advanced prompting techniques

Lessons Learned

  • AI systems require structured workflows, not just models
  • Prompt quality impacts both performance and cost
  • System integration is key in real-world applications
