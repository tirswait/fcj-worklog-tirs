## Month 1–2: AWS Fundamentals

**Estimated Budget:** $50–70

### Learning Objectives

* Understand AWS global infrastructure
* Master core AWS services (EC2, S3, IAM, VPC)
* Learn basic security and networking concepts
* Get familiar with AWS CLI and SDK

---

### Curriculum

#### Week 1 – Getting Started

* Complete 5 credit-earning tasks
* Set up billing alerts and AWS budgets
* Explore AWS Management Console
* Create the first EC2 instance
* Hands-on: Deploy a simple web server

#### Week 2 – Storage & Databases

* Amazon S3 buckets and lifecycle policies
* EBS volumes and snapshots
* Amazon RDS setup and management
* Hands-on: Build a file upload system

#### Week 3 – Networking

* VPC concepts and configuration
* Security Groups vs Network ACLs
* Internet Gateway and NAT Gateway
* Hands-on: Multi-tier network architecture

#### Week 4 – Identity & Security

* IAM users, groups, and roles
* Policies and permission management
* Multi-Factor Authentication (MFA)
* Hands-on: Secure multi-user AWS environment

---

### Projects

**Project 1: Personal Blog Website**

* EC2 + RDS + S3
* Estimated Cost: ~$25/month

**Project 2: File Sharing System**

* S3 + Lambda + API Gateway
* Estimated Cost: ~$10/month

## Certification Path

**AWS Certified Cloud Practitioner**

* Target Time: End of Month 2
* Exam Fee: $100
* Preparation Time: 2–3 weeks
* Expected Success Rate: ~90%

**AWS Solutions Architect – Associate**

* Target Time: Month 4–5
* Exam Fee: $150
* Preparation Time: 4–6 weeks
* Expected Success Rate: ~70%

**AWS Solutions Architect – Professional**

* Target Time: Month 6+
* Exam Fee: $300
* Preparation Time: 8–12 weeks
* Expected Success Rate: ~50%
