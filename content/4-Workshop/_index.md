# Authentication and Access Control using AWS IAM

---

#### Overview

AWS Identity and Access Management (IAM) is a service that enables secure control of access to AWS resources.

In this workshop, you will learn how to:
- Create an IAM user
- Assign permissions using policies
- Verify access control using the AWS IAM dashboard

This workflow simulates a basic **authentication and authorization system**, where:
- Authentication defines **who can access**
- Authorization defines **what actions they can perform**

It also demonstrates **Role-Based Access Control (RBAC)** using AWS IAM policies.
