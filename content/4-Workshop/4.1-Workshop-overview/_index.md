# Workshop Overview

This workshop demonstrates how to use AWS IAM to control access to cloud resources.

You will learn:
- How to create IAM users
- How to assign permissions using groups and policies
- How to test access using real AWS services

This workshop simulates a real-world access control system using RBAC.
