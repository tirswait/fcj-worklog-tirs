In this section, we will create an **IAM User**, assign the user to an existing group.  

## Objectives

After completing this section, you will be able to:

- Create an **IAM User**
- Assign the user to an existing group

---

## Step 1: Create IAM User & Add to Group

Go to:

- **IAM**
- **Users**
- **Create user**

At this step:

- Enter username: **dev-2**
- Select: **Provide user access to the AWS Management Console**
- Choose: **Add user to group**
- Select group: **test-policy**

![user](/images/4-Workshop/4.2-Authentication-and-RBAC/4.png)

![user](/images/4-Workshop/4.2-Authentication-and-RBAC/5.png)

---

## Step 2: Complete User Creation

- After creation, the user **dev-2** will appear in the user list.

![user](/images/4-Workshop/4.2-Authentication-and-RBAC/6.png)

---

## Step 3: Enable Console Access and Get Login Info

Go to:

- **IAM** > **Users** > **dev-2**

Do:

- Enable **Console access**

![login](/images/4-Workshop/4.2-Authentication-and-RBAC/7.png)

- Choose **Autogenerated password**, **Custom password** or **User sign-in with new password**

![login](/images/4-Workshop/4.2-Authentication-and-RBAC/8.png)

- Copy **Sign-in URL** (this will use for User sign-in), username, and password

![login](/images/4-Workshop/4.2-Authentication-and-RBAC/9.png)
