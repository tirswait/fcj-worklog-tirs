### Week 7 Objectives:

* Understand container concepts and Docker fundamentals.
* Learn how to build and manage container images.
* Explore Amazon ECS (Elastic Container Service).
* Work with Amazon ECR (Elastic Container Registry).
* Deploy a containerized web application on AWS.
