### Week 5 Objectives:

* Understand serverless architecture and its benefits.
* Learn how to create and manage AWS Lambda functions.
* Explore event triggers for Lambda.
* Configure API Gateway to build APIs.
* Work with DynamoDB as a NoSQL database.
* Build a REST API backend using serverless architecture.
